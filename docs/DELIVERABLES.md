# Deliverables & Specs

| Deliverable |                       Format | Notes                                  |
| ----------- | ---------------------------: | -------------------------------------- |
| Full video  |        MP4, 1920x1080, H.264 | high-bitrate master (e.g., 10–20 Mbps) |
| Audio       |       MP3 96–192 kbps or WAV | Podcast: -14 LUFS target               |
| Clips       | MP4, 1080x1080 / 9:16 shorts | 15–90s, captions                       |
| Captions    |                    VTT / SRT | time-synced; upload to YT              |
| Thumbnail   |             1280x720 JPG/PNG | consistent templated design            |
| Transcript  |                  .md or .vtt | searchable on site                     |
