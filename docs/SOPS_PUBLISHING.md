# SOP - Publishing and Release

## Purpose
Define release steps for full episodes, clips, and website updates.

## Inputs
- Final masters, captions, thumbnails, metadata, and approved copy.

## Outputs
- Published episode, website updates, and scheduled social posts.

## Checklist - YouTube Upload
- [ ] Upload full episode with title, description, tags, and thumbnail.
- [ ] Add chapters and sponsor timestamps.
- [ ] Set privacy to Scheduled or Public as required.
- [ ] Validate video processing and playback.

## Checklist - Website Update
- [ ] Publish episode metadata and transcript to the site.
- [ ] Confirm episode page renders and links work.
- [ ] Update RSS feed if applicable.

## Checklist - Social Release
- [ ] Schedule or publish social posts for each platform.
- [ ] Validate the first post goes live on time.
- [ ] Save post IDs and URLs for reporting.

## Checklist - Post-Release Verification
- [ ] Confirm episode is live on YouTube and any podcast host.
- [ ] Verify social posts are visible and accurate.
- [ ] Monitor for comments or platform errors in the first hour.
