# Jared's Not Funny — Production Repository

This repo contains production workflows, SOPs, scripts, and website scaffolding to support Jared Christianson's podcast and related content creation.

Quick links

- `docs/` — SOPs and checklists
- `raw_videos/` — raw footage ingests
- `edited/` — edited episodes and deliverables
- `assets/` — thumbnails, overlays, project files
- `scripts/` — automation scripts (ingest, transcode, clip generation)
- `website/` — static site for episodes, tour dates, SEO
- `.github/` — issue templates, labels, actions

Read `ROADMAP.md` for milestones and deliverables.
